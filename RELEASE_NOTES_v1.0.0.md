# 服饰识别 v1.0.0 - 首次发布

## 📱 应用简介

**服饰识别** 是一款面向 50 类服饰的离线识别 Android 应用，主打"简约、稳定、可落地"。应用采用 Material Design 3 现代化界面设计，提供流畅的用户体验。

## ✨ 核心特性

### 识别功能
- ✅ **50 类服饰识别**：支持 DeepFashion 数据集的完整 50 个类别
- ✅ **离线 ONNX 推理**：无需网络连接，完全本地化识别
- ✅ **实时相机识别**：支持前后摄像头切换，实时识别服装类型
- ✅ **相册选择**：支持从相册选择图片进行识别

### 用户体验
- ✅ **Material Design 3**：现代化 UI 设计，支持深色模式
- ✅ **多语言支持**：中文/English 界面切换，设置即时生效
- ✅ **手势操作**：双指缩放、双击对焦（黄色对焦框）
- ✅ **性能优化**：多线程推理支持，性能优先设置

### 历史记录
- ✅ **结果保存**：识别结果一键保存到历史记录
- ✅ **智能筛选**：支持按类别、置信度、时间筛选
- ✅ **搜索功能**：快速搜索历史记录
- ✅ **分享功能**：支持分享文字信息和卡片图片

### 设置功能
- ✅ **语言切换**：中文/English，即时生效
- ✅ **性能优先**：多线程推理开关，即时生效
- ✅ **主题模式**：跟随系统/浅色/深色
- ✅ **结果展示密度**：自定义结果展示方式
- ✅ **默认摄像头**：选择默认使用的摄像头

## 🎯 模型性能

### 训练状态
- **训练完成**：32/100 epochs（Early Stopping 触发）
- **最佳验证准确率**：61.55%（Epoch 22）
- **最终训练准确率**：74.29%
- **最终验证准确率**：60.40%
- **训练-验证差距**：13.89%（显著改善）

### 防过拟合措施
- 增强数据增强（随机旋转、颜色变换、仿射变换）
- 增强正则化（Dropout 0.6、权重衰减 2e-4、标签平滑 0.1）
- Early Stopping（连续 10 个 epoch 无提升自动停止）
- GPU 加速训练（混合精度训练）

### 技术架构
- **骨干网络**：ResNet18
- **迁移学习**：ImageNet 预训练权重
- **推理引擎**：ONNX Runtime
- **开发语言**：Kotlin
- **最小 SDK**：Android API 24+

## 📦 安装使用

### 构建应用

1. **导入工程**
   ```powershell
   # 使用 Android Studio 打开 DeepFashionClassifier 目录
   ```

2. **放置模型**
   - 确保 ONNX 模型位于：`app/src/main/assets/models/deepfashion_classifier.onnx`
   - 模型文件需要单独下载（由于体积限制，不在仓库中）

3. **构建安装**
   ```powershell
   cd "DeepFashionClassifier"
   .\gradlew.bat assembleDebug
   ```

4. **生成图标**（可选）
   ```bash
   python scripts/generate_launcher_icons.py --scale 0.60
   ```

### 训练模型

如需重新训练模型：

```bash
# 安装依赖
pip install torch torchvision onnx onnxruntime pillow

# 训练模型
python scripts/train_deepfashion_complete.py

# 导出 ONNX 模型并更新应用
python scripts/update_model_for_android.py
```

## 📝 已知问题

1. 模型准确率仍有提升空间：当前验证准确率 61.55%，可继续优化
2. Android 权限：需要手动授权相机和存储权限
3. 样本不平衡：部分类别样本数量极少，可能影响识别准确率
